```

   ✓ 月額固定で初期投資がない
   ✓ 採点削減メリットは確実
   ✓ 最悪のケースでも年間¥280Kの黒字

   ただし、以下の場合は効果が薄れます：
   ❌ 講師が「AI採点を信頼しない」で手採点を継続
   ❌ 生徒にUI説明を十分にしない
   ❌ サーバー障害が多い環境

   → 対策: 導入前のトレーニング、定期的なメンテナンス
```

### Q: 年間契約 vs 月間契約どちらがおすすめですか？
```
A: 小規模塾: 月間契約から開始（柔軟性重視）
   中規模以上: 年間契約（15%割引）

   理由：
   ・小規模は試行期間が必要
   ・中規模は安定稼働が見込めるため割引活用
```
