# EigoMaster 営業資料パッケージ

## 📚 資料一覧

### 1️⃣ **デモシナリオ文書**
**ファイル**: `01_DEMO_SCENARIO.md`
**用途**: 営業デモの実施手順書
**対象者**: 営業担当者、デモ実施者
**分量**: 約25ページ
**所要読了時間**: 30分

#### 内容
- デモの全体フロー（30分）
- 5つの主要シーン（環境準備～質問対応）
- ユースケース1: 講師視点（リスニング・シャドーイング）
- ユースケース2: 生徒視点（学習管理・進捗追跡）
- ダッシュボード・分析機能の実演
- FAQ コーナー（よくある質問への応答話術）
- デモ用スクリーンショット配置位置
- 講師向けオペレーション手順書
- デモビデオシナリオ（3分）
- デモ前チェックリスト
- デモ後のフォローアップ資料

#### 使用シーン
```
営業デモの実施
  ├─ Day 1: 初回訪問
  │  └─ 20-30分のデモンストレーション
  ├─ Day 2: 導入検討企業向け
  │  └─ 実際のアカウントでの操作デモ
  └─ Day 3: 意思決定者（経営者）向け
     └─ ビジネスメリット中心のデモ
```

  - ライティング添削機能

  ├─ プレゼンテーション
  │  └─ スライド化して説明

     └─ 詳細なメリット・料金を確認

### 4️⃣ **ROI試算シート**
**ファイル**: `04_ROI_CALCULATOR.md`
**用途**: 経営判断・予算組資料
**対象者**: 経営者、CFO、経営企画責任者
**分量**: 約35ページ
**所要読了時間**: 15分（概要）/ 45分（詳細）

#### 内容
- シナリオ別ROI試算
  - シナリオA: 小規模塾（30人）
  - シナリオB: 中規模塾（60人）
  - シナリオC: 大規模塾（120人）
- 詳細コスト分析
  - 採点業務の時間削減詳細
  - 新しく作り出せた時間の活用
- 売上向上シミュレーション
  - パラメータ分析（感度分析）
  - 最悪シナリオ～最良シナリオ
- 経営者向けの主要指標（3つのKPI）
- 月単位の経営シミュレーション（初月～12月）
- 次フェーズへの拡張（ROIさらに向上）
- 競合との料金比較
- 導入前の質問チェック
- 3年間の長期シミュレーション
- FAQ（導入判断用）

#### 使用シーン
```
経営判断フェーズ
  ├─ 予備調査
  │  └─ 「本当に投資対効果があるか？」確認

  │  └─ ROI数字を使った経営判断

**用途**: IT管理者・情報セキュリティ責任者向け資料

   - エンドポイント一覧
   - 認証フロー（OAuth 2.0）

   - 暗号化標準（転送中・保存中）

   - 個人情報取扱方針
   - 定期監査

4. パフォーマンス指標
   - レスポンスタイム
   - スケーラビリティ
   - 可用性（SLA）

5. ネットワーク設定ガイド
   - ファイアウォール設定
   - DNS設定
   - トラブルシューティング

6. ディザスタリカバリ & バックアップ
   - バックアップ戦略
   - インシデント対応

7. システム監視 & ロギング
   - 監視対象
   - ログ記録・保持期間

8. テクニカルサポート
   - 提供項目
   - 連絡方法

9. コンプライアンス & 認証
   - 国際標準
   - 法的要件

10. FAQ（IT管理者向け）
11. 管理者用チェックリスト

#### 使用シーン
```
IT環境構築フェーズ
  ├─ ネットワーク設定
  │  └─ ファイアウォール・DNS設定
  ├─ セキュリティ確認
  │  └─ 暗号化・データ保護を確認
  ├─ インフラ準備
  │  └─ 帯域幅・デバイス確認
  └─ 運用開始

## ✅ よくある質問（営業チーム向け）

### Q: 資料のカスタマイズは可能ですか？
```
A: はい。以下の範囲でカスタマイズ可能です：
  ✓ 相手先企業名の追加
  ✓ 地域別の事例追加
  ✓ 料金・契約条件の修正
  ✗ 基本的なコンセプト・数字の変更

営業事務まで相談してください。
```

### Q: 資料は何か月有効ですか？
```
A: 基本的には有効期限なしですが、以下は定期更新が必要です：
  ✓ ROI試算（毎月）
  ✓ セキュリティ情報（毎年）
  ✓ 技術仕様（毎年）

古い情報で営業すると、信頼失失のため注意してください。
```
