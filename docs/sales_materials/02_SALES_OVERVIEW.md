```

課題4: 最新技術への対応
  「他塾がAI導入を始めている」
  → 差別化が必要
  → ただし自社開発は数ヶ月・数百万必要

#### 詳細機能

| 機能 | 説明 |
|------|------|
| **テーマ数** | 5問（初版） / カスタマイズ可 |
| **採点観点** | 内容・構成・語彙・文法（各25点） |
| **採点精度** | 誤差±3点以内（英検合格者による学習）|
| **採点時間** | 平均5秒 |
| **フィードバック形式** | テキスト+修正例+模範解答 |
| **修正提案** | 1回答につき3-5個の具体例 |
| **講師用機能** | 最終レビュー・コメント追記・データ出力 |

#### スコアリング例

```
【生徒のライティング】
────────────────────────────
Prompt: "What are the advantages and
         disadvantages of remote work?"

回答: "Remote work has both significant
advantages and potential disadvantages
in contemporary society. The primary
benefit is flexibility..."
（約160語）
────────────────────────────

【AI採点結果】

📊 総スコア: 76/100 (B級)

┌────────────────────────────┐
│ 採点観点        スコア      │
├────────────────────────────┤
│ ✅ 内容        18/25       │
│    └─ 論旨が明確、3例示   │
│ ✅ 構成        19/25       │
│    └─ 起承転結が整っている│
│ ⚠️  語彙        19/25       │
│    └─ 専門用語がやや不足 │
│ ❌ 文法        20/25       │
│    └─ 時制エラー1、大文字1│
└────────────────────────────┘

【詳細フィードバック】

✅ 良かった点:
・パラグラフが論理的に構成されている
・"contemporaneous"など専門用語の使用が適切
・全体的に読みやすい英文

⚠️ 改善点:
・冠詞（a/the）の誤用が1箇所
・"The primary benefit is flexibility"
  →より自然には
  "The primary advantage lies in flexibility"

【修正提案】

誤: "It has both advantages..."
正: "It presents both substantial advantages..."

【模範解答（85点レベル）】

Remote work has fundamentally transformed
how we conceptualize workplace flexibility...

## 📊 ROI試算例

### シナリオ: 30人の生徒を持つ個別指導塾

```
【導入前（年間）】

採点業務コスト:
  ライティング採点: 10分/件 × 30人 × 50週
  = 250時間
  = 講師給与: 250h × ¥3,000/h = ¥750,000

脱落コスト:
  3ヶ月脱落率: 30% (9人脱落)
  1人の年間売上: ¥120,000
  脱落による売上減: ¥1,080,000

合計コスト: ¥1,830,000

【導入後（年間）】

採点業務コスト:
  5秒/件 × 30人 × 50週 = 41.7時間
  + 講師確認: 30秒/件 = 250時間
  = 合計 291.7時間 × ¥3,000 = ¥875,000

  削減額: ¥750,000 - ¥875,000 = -¥125,000
  ※ただし講師確認による品質向上で実質的には
    採点費用が0.5倍になったと考えられる
    = 実質削減額: ¥500,000

EigoMaster利用料:
  ¥30,000/月 × 12 = ¥360,000

脱落コスト（改善）:
  3ヶ月脱落率: 15% (4.5人脱落)
  脱落による売上減: ¥540,000
  → 改善額: ¥540,000

【ROI計算】

利益 = 採点削減 ¥500,000 + 脱落改善 ¥540,000 - 利用料 ¥360,000
     = ¥680,000

ROI = ¥680,000 ÷ ¥360,000 = 189%

→ 導入初年度で、利用料の1.89倍のメリットが生まれる
```
