## 📱 よくある問題と対応

### ログイン関連

| 問題 | 原因 | 対応 |
|------|------|------|
| **パスワード忘却** | 初期パスワードを覚えていない | パスワードリセット機能を使用 |
| **メールアドレス誤記** | CSVの誤入力 | 管理画面から修正 |
| **初回メール未受信** | スパムフォルダに入った | スパムフォルダを確認 |
| **2段階認証失敗** | 設定エラー | QRコードを再取得 |

### 音声関連

| 問題 | 原因 | 対応 |
|------|------|------|
| **音声が出ない** | スピーカー未接続 / マナーモード | 音量を確認、マナーモード解除 |
| **音声の遅延** | ネットワーク遅延 | Wi-Fi接続確認、キャッシュクリア |
| **再生が途中で止まる** | バッファアンダーラン | 5秒待機後、再度再生 |

### 回答・採点関連

| 問題 | 原因 | 対応 |
|------|------|------|
| **回答が保存されない** | キャッシュエラー | ページリロード、ブラウザ再起動 |
| **AI採点が遅い** | サーバー負荷 | 30分待機、それでもダメなら再提出 |
| **スコアが不正確** | 標準と異なる回答 | 講師が最終確認・調整 |

### デバイス関連

| 問題 | 原因 | 対応 |
|------|------|------|
| **タブレットで崩れて見える** | 画面サイズ対応不足 | 横向きで使用、或いはPC版を利用 |
| **バッテリー消費が早い** | 音声再生の負荷 | 明るさ調整、アプリ定期終了 |

## 📞 サポート体制の詳細

### サポート受付方法（複数の選択肢）

```
【メールサポート】
- 宛先: support@eigomaster.example.com
- 応答時間: 24時間以内
- 対応内容: 全トラブル

【電話サポート】
- 番号: 0120-xxx-xxxx
- 受付時間: 平日 9:00-18:00
- 対応内容: 緊急トラブルのみ

【チケットシステム】
- URL: support.eigomaster.example.com
- 応答時間: 2営業日以内
- 対応内容: 機能要望・改善提案

【Slack コミュニティ】
- ワークスペース: eigomaster-jp
- 参加: 招待リンクを受け取る
- 対応時間: 営業時間内
- 対応内容: 質問・情報共有
```

### サポート優先度

```
【P1: 緊急（1時間以内対応）】
- 全生徒がログインできない
- データが全て失われた
- セキュリティ侵害の可能性

【P2: 高（4時間以内対応）】
- 一部の生徒がログイン不可
- 採点機能が動作しない
- 大量のバグ報告

【P3: 中（24時間以内対応）】
- 個別トラブル
- UI/UX改善要望
- 機能についての質問

【P4: 低（3営業日以内対応）】
- マニュアルの誤り
- 細かい表示バグ
- 長期的な改善提案
```
