# 🎯 Shadowing Feedback システム改善 - Shadoten風の詳細添削実装

## 📋 概要

EigoMasterのシャドーイング添削機能を「Shadoten」のような詳細で実用的なフィードバックシステムに改善しました。

### Before（改善前）
```
ラウンド1の音読評価です。もっと正確に話しましょう。
```

### After（改善後）
```
【ラウンド1の詳細評価】

✓ 正確性: 60% の正確さです。もう少し細部に注意を払いましょう。
特に「original」の部分を確認してください。

🎵 リズム: リズムはまあまあです。もう少し速度を調整するといいでしょう。

🗣️ 発音: 大体の発音は正しいですが、いくつかの音が不正確です。

💡 アドバイス: 着実に上達しています。もう少し練習を重ねると、もっと良くなるでしょう。
ラウンド3までは基本を固める段階です。焦らず丁寧に。
```

---

## 🚀 実装内容

### 1️⃣ AIスコアリングサービスの改善
**ファイル**: `/src/lib/aiScoringService.ts`

#### 変更点:
- **ダミースコア生成の高度化** (`generateDummyScore()`)
  - スクリプトと転写内容を word-by-word で比較
  - 一致率（Accuracy Rate）を計算して精度に反映
  - スコアが上がるほど具体的なフィードバック生成

#### 新しい関数群:
- `findDifferences()` - スクリプトと異なる単語を検出、修正提案を生成
  - 最初の3つの異なる単語を見つける
  - 単語数の不一致を検出
  - 実践的な改善提案を提供

- `generateSpecificFeedback()` - スコアに基づいた具体的なフィードバック生成
  - 正確性に基づいたコメント（8以上/6以上/4以上/4未満で異なる）
  - リズムに関する段階的なアドバイス
  - 発音に関する具体的な指導
  - ラウンド進行に応じた励まし
  - 全体的なアドバイス（初級/中級/上級向け）

---

### 2️⃣ DetailedShadowingFeedback コンポーネント
**ファイル**: `/src/components/DetailedShadowingFeedback.tsx`（新規作成）

#### 機能:

**a) スコア表示セクション**
```
正確性: [████████░░] 8.0/10
リズム: [██████░░░░] 6.5/10
発音:   [███████░░░] 7.0/10
```
- プログレスバーで視覚的に表示
- スコアに応じて色が変わる（緑→オレンジ→赤）

**b) 単語ごとのフィードバック**
```
✓ correct  ✗ incorrect  △ weak
```
- スクリプトの各単語を色分け（青/赤/灰色）
- タップ可能で詳細モーダルを開く

**c) 詳細フィードバック モーダル**

ユーザーが単語をタップすると以下の情報が表示:
- **単語表示**: 対象の単語を強調表示
- **タグ**: Pronunciation / Linking / Rhythm / Grammar / Meaning
- **評価**: ✓正確 / ✗不正確 / △弱い
- **あなたの発音**: 聞こえた内容（カタカナ）
- **正しい発音**: 正しい発音のカタカナ表記
- **何が間違っていたのか**: 技術的な説明
- **改善のコツ**: 実践的なアドバイス

## 📊 フィードバック生成のロジック

### 正確性スコア別のメッセージ
```javascript
8.0以上 → "スクリプトにほぼ完全に一致しています。素晴らしい！"
6.0-8.0 → "もう少し細部に注意を払いましょう"
4.0-6.0 → "いくつかの単語の発音に改善の余地があります"
4.0未満 → "もう一度ゆっくり聞いて、丁寧に繰り返してください"
```

### リズムスコア別のメッセージ
```javascript
7.0以上 → "流暢で自然なリズムです"
5.0-7.0 → "もう少し速度を調整するといいでしょう"
5.0未満 → "スクリプトの音声を何度も聞いて、パターンを学びましょう"
```

### 発音スコア別のメッセージ
```javascript
7.0以上 → "個々の単語の発音が正確です"
5.0-7.0 → "いくつかの音が不正確です"
5.0未満 → "各単語を個別に練習してみてください"
```

### ラウンド進行別の励まし
```javascript
Round 1-3: "基本を固める段階です。焦らず丁寧に"
Round 4-5: "後半に向けて速度を上げていく準備ができてきましたね"
Round 6-7: "流暢さを意識してみてください"
```

## 📝 参考資料

**Shadoten の特徴**（実装時の参考）
- ✅ 毎日プロが添削してくれる（実装: AI による自動添削）
- ✅ 自分では気づけない音のズレを指摘（実装: word-level 分析）
- ✅ 「お金払ってるからサボれない」というプレッシャー（実装: ゲーミフィケーション）
- ✅ 具体的でありがたい添削（実装: 詳細なモーダルフィードバック）

**実装の利点**
- 無料で Shadoten 相当の機能を提供
- オンデマンドでアクセス可能
- パーソナライズされたフィードバック
- 継続学習への強いモチベーション
