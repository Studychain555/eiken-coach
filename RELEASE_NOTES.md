# EigoMaster リリースノート

## v1.0.0 - 初回本番リリース（2026-03-26予定）

### 🎉 概要

EigoMasterは、英検準1級受験者向けの統合学習アプリです。リスニング、単語、シャドーイング、ライティング添削の4つの機能で、効率的な英語学習をサポートします。

**リリース日**: 2026年3月26日（予定）
**バージョン**: 1.0.0
**ビルド番号**: iOS 1, Android 1

---

## ✨ 新機能

### 1. 英単語テスト
- **説明**: 英検準1級の重要単語（2,000語以上）を4択問題で学習
- **復習アルゴリズム**: SM-2（Spaced Repetition）で効率的な復習をサポート
- **進捗管理**: ステージごとの進捗バーで学習状況を可視化
- **統計表示**: 学習時間、正答率、レベル等をダッシュボード表示

### 2. リスニング問題
- **コンテンツ**: 英検準1級レベルの音声問題（100問以上）
- **音声再生**: HTML5 Audio API + expo-av で安定した再生
- **再生速度制御**: 0.5x～1.5x の速度調整機能
- **スクリプト表示**: 日本語訳とともにスクリプトを表示
- **エラーハンドリング**: CORS対応、自動リトライ、タイムアウト処理

### 3. シャドーイング練習
- **7ラウンド制御**: 聞く→何度も繰り返して録音
- **音声認識**: Whisper API による自動文字起こし
- **AI採点**: Claude API による自動添削スコアリング
- **フィードバック**: 音韻、速度、イントネーション等の改善点を提示
- **グラフ表示**: ラウンド別のスコア推移を可視化

### 4. ライティング添削
- **問題選択**: 複数のライティング課題から選択
- **入力方法**: テキスト入力またはカメラで手書き画像撮影
- **OCR**: 手書き画像を自動認識（Google Vision API使用時）
- **AI採点**: Claude API による自動採点（文法、語彙、流暢性、構成）
- **フィードバック**: 詳細な採点結果と改善点、模範解答を提示

### 5. 認証・プロフィール
- **Supabase 統合**: ユーザー認証、データベース管理
- **プロフィール管理**: ユーザー情報（名前、メール、学習目標）の設定
- **パスワードリセット**: メール確認によるセキュアなパスワード変更
- **プロフィール写真**: (実装予定)

### 6. ダッシュボード
- **統計表示**: 本日/本週の学習時間、正答率、学習ストリーク
- **カテゴリ別進捗**: 単語、リスニング、シャドーイング、ライティングの進捗
- **グラフ表示**: 週別、月別の学習時間推移を折れ線グラフ表示
- **クイックアクション**: 「今すぐ学習」「復習」ボタン

## ❓ FAQ

### Q. 有料ですか？

A. v1.0.0は無料です。将来、プレミアムプランを検討する可能性があります。

### Q. オフライン使用できますか？

A. v1.0.0ではオンライン接続が必須です。v1.1でオフラインモード実装予定です。

### Q. 音声認識に対応していますか？

A. シャドーイングとライティングで音声認識機能を搭載しています。（Whisper API使用）

### Q. サポートは日本語ですか？

A. はい。UI、メッセージ、ドキュメントはすべて日本語です。

### Q. iPhone のみ？ Android は？

A. iOS と Android の両方に対応しています。

### Q. ダークモードは？

A. はい。システムの設定に自動的に従います。
