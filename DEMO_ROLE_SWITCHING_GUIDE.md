## 🔄 ロール切り替え方法

**生徒から講師に切り替え:**
1. F5 キーで画面をリロード（または Cmd+R）
2. ロール選択画面が再表示される
3. 「講師として開始」を選択

**講師から生徒に切り替え:**
1. F5 キーで画面をリロード
2. 「生徒として開始」を選択

## 🛠️ 技術実装

### 修正ファイル
**File**: `/Users/80dr/eigomaster/app/demo.tsx`

**追加内容**:
1. ロール選択状態管理
```typescript
const [selectedRole, setSelectedRole] = useState<'student' | 'teacher' | null>(null);
```

2. ロール選択ハンドラ
```typescript
const handleSelectRole = (role: 'student' | 'teacher') => {
  useAuthStore.setState({ role });
  setSelectedRole(role);
};
```

3. 条件付きレンダリング
```typescript
if (!selectedRole) {
  return <RoleSelectScreen />;
}
return <DemoGameScreen />;
```

4. 新規スタイル
```
- roleSelectContainer: ロール選択画面のコンテナ
- roleButton: ロール選択ボタン
- studentButton / teacherButton: 各ロール用スタイル
- infoBox: ヒント表示ボックス
```

## 🐛 トラブルシューティング

**Q: ロール選択画面が表示されない**
A: キャッシュをクリア（Ctrl+Shift+Delete）してからアクセス

**Q: 「生徒として開始」を選択しても何も起きない**
A: ブラウザコンソール（F12）でエラーを確認

**Q: タブの表示が変わらない**
A: F5 で完全にリロードしてください（ブラウザキャッシュの影響）

**Q: 講師タブが表示されているはずなのに見えない**
A: TabLayout が useAuthStore から role を正しく読み込んでいるか確認
