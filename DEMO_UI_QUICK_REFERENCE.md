## 🎯 デモポイント（塾に訴求する要素）

### 「スタディサプリのような洗練された管理画面」
1. **左サイドバー**
   - 「紺色でプロフェッショナルな見た目」
   - 「メニューがシンプルにまとまっている」
   - 「クリックすると、各セクションがスムーズに切り替わる」

2. **右サイドバー統計**
   - 「一目で重要な情報が見える」
   - 「生徒数、課題数、進捗率、学習時間が即座に把握可能」
   - 「学習状況が色（緑/黄/赤）で直感的に理解できる」

3. **メインコンテンツ**
   - 「クラス全体の統計情報が上部に集約」
   - 「最近の課題が見やすくカード形式」
   - 「成績優秀者が一目瞭然（ランク表示）」

## 🔄 デモ操作フロー（推奨）

## 💡 説明ポイント

### 「塾経営者向け」のメリット説明
```
1️⃣ 「効率化」
   「各タブに分けることで、必要な情報だけをサッと確認できます。
    従来のタブ切り替え方式より直感的です。」

2️⃣ 「視認性」
   「右側のパネルが常に見えているので、
    クラス全体の学習状況をいつでも把握できます。」

3️⃣ 「プロフェッショナル感」
   「スタディサプリのような、ハイエンドなSaaSのような見た目で、
    生徒・保護者からの信頼感が向上します。」

4️⃣ 「拡張性」
   「今後、AI分析や自動採点機能も
    このインターフェース上に統合可能です。」
```

## 🎬 よくある質問への回答例

**Q: 「スタディサプリのような」というのは、実際に同じ企業が開発したんですか？**
```
A: いいえ。EigoMasterは独自開発の塾向けシステムです。
   スタディサプリの優れたUX/UIを参考にしながら、
   EigoMasterの特性に合わせて最適化しています。
```

**Q: 「今後AI分析機能が追加できる」とのことですが、いつ追加ですか？**
```
A: ロードマップとしては Q2 2026 を目指しています。
   まずは現在の基本機能を堅実に運用いただき、
   フィードバックに基づいて拡張予定です。
```

**Q: モバイルでも同じように見えますか？**
```
A: モバイルではレスポンシブ設計で、
   サイドバーがスライドイン方式に自動切り替わります。
   （あればモバイル画面も見せると良い）
```
