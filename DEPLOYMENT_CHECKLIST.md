## 📦 ビルドとリリース

### ローカルビルド
```bash
# Web テスト
npm run web

# iOS シミュレータ
npm run ios

# Android エミュレータ
npm run android
```

### EAS ビルド（Expo サービス）
```bash
# インストール
npm install -g eas-cli

# ログイン
eas login

# ビルド設定
eas build --platform ios
eas build --platform android
```

### TestFlight / 内部テスト配信
```bash
# iOS: TestFlight
eas build --platform ios --release-channel production

# Android: Google Play 内部テスト
eas build --platform android --release-channel production
```
