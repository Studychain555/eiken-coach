## 🐛 エラーハンドリング・エラーメッセージ

### 表示内容確認

- [ ] すべてのエラーメッセージが日本語
- [ ] エラーメッセージが明確・理解しやすい
- [ ] エラーメッセージに解決方法が示されている

### エラータイプ別対応

| エラー | 表示内容 | 対応方法 |
|--------|--------|--------|
| ネットワークエラー | 「ネットワークに接続してください」 | Wi-Fi/モバイルネットワークを確認 |
| API エラー (4xx) | 「リクエストエラーが発生しました」 | 入力内容を確認・再試行 |
| API エラー (5xx) | 「サーバーエラーが発生しました」 | しばらく待ってから再試行 |
| 音声読込エラー | 「音声ファイルが読込めません」 | 再生ボタンで再試行 |
| タイムアウト | 「リクエストがタイムアウトしました」 | 再試行ボタンで再接続 |
| 認証エラー | 「ログインの認証に失敗しました」 | ログイン画面に戻る |

### Sentry への自動報告

- [ ] エラーが Sentry に自動報告される
- [ ] Sentry ダッシュボードでエラー内容が確認できる
- [ ] エラーの頻度・デバイス情報が記録される
