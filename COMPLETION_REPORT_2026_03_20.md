### Phase 9: 学習機能

**ニーモニック例**:
```typescript
// "ability" → "Able + Bility"
technique: "acronym"
effectiveness: 4/5

// "absolute" → Ab-(away) + solute(loosened)
technique: "etymology"
effectiveness: 5/5
```

**SM2+ アルゴリズム**:
```
復習間隔: 1日 → 3日 → 7日 → 14日 → 30日
難度係数: 1.3 ~ 2.5 (動的調整)
取得成功率: 次回復習の確度を予測

計算式:
EF = max(1.3, EF + (0.1 - (5-q) * (0.08 + (5-q) * 0.02)))
```

**学習パターン分析**:
```
入力: メトリクス配列
出力:
  - 強み (e.g., "高い正答率")
  - 弱み (e.g., "反応遅い")
  - 推奨焦点
  - 学習習慣評価
```
