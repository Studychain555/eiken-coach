## 👀 何を確認すべきか

### ✅ 確認ポイント

1. **フィードバックの詳細さ**
   ```
   Before: 「もっと正確に話しましょう。」（1行）
   After:  「正確性: 60%。特に「studying」を...」（複数段落）

   → 5倍以上の詳細さ ✨
   ```

2. **スコアの内訳**
   ```
   正確性 / リズム / 発音 の3つが独立して表示
   → それぞれの強み/弱点が明確に
   ```

3. **単語ごとの分析**
   ```
   ✓✗△ で色分けされた単語リスト
   タップで詳細モーダル表示
   → 何が悪かったのか一目瞭然
   ```

4. **実践的なアドバイス**
   ```
   「/ɪŋ/ の音を意識する」
   「鼻音で終わる」
   → すぐに応用できる
   ```

5. **UI/UX の改善**
   ```
   · 温かい自然な色（NaturalColors パレット）
   · 展開可能なカード設計
   · インタラクティブなモーダル
   → Shadoten のような高品質感
   ```

## 🎯 よくある質問

**Q: 毎回同じフィードバックが出るのか？**
A: いいえ。スクリプトと転写を比較するので、異なる音読には異なるフィードバックが出ます。

**Q: モーダルが表示されない場合は？**
A: DetailedShadowingFeedback コンポーネントが正しく表示されている確認してください。console を確認し、エラーログを探してください。

**Q: 単語ごとのタグ（Pronunciation / Linking など）は？**
A: デモでは自動割り当てされています。本番では Claude API で自動判定されます。

**Q: これはいつ本番になるのか？**
A: 手動テスト後、PR を作成して本番マージできます。
