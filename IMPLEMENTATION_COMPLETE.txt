================================================================================
                         セキュリティ設定値
================================================================================

【タイムアウト】
- アクセストークン: 15分
- リフレッシュトークン: 7日
- セッション: 30分
- API タイムアウト: 30秒

【レート制限】
- ログイン試行: 5回/分
- API リクエスト: 100回/分
- 最大リクエストサイズ: 10MB

【パスワード】
- 最小長: 12文字
- 大文字・小文字・数字・特殊文字: 必須
- パターン: /^(?=.*[a-z])(?=.*[A-Z])(?=.*\d)(?=.*[@$!%*?&])[A-Za-z\d@$!%*?&]{12,}$/

【暗号化】
- アルゴリズム: AES-256-GCM
- キーサイズ: 32バイト
- IV: 16バイト (ランダム)
- 認証タグ: 16バイト
