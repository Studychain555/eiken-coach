# 塾管理者向けスタートガイド

**対象**: 塾長・教務主任・システム管理者
**内容**: ユーザー登録からシステム運用までの完全手順
**難易度**: 初心者向け（技術知識不要）
**所要時間**: 1時間で全セットアップ完了

---

## 第1章: 塾管理者のログイン設定（10分）

### 1-1. 初回ログイン

**ステップ1**: ブラウザで以下のURLにアクセス

```
https://cram.example.jp
```

**ステップ2**: ログイン画面が表示される

```
┌─────────────────────────────────────┐
│   eigomaster ログイン                │
│                                     │
│ メールアドレス: [            ]      │
│ パスワード:     [          ]        │
│ ☐ パスワードを保存                  │
│                                     │
│            [  ログイン  ]           │
│                                     │
│ 初めての方？[パスワード再設定]      │
└─────────────────────────────────────┘
```

**ステップ3**: 管理者メールアドレスとパスワードを入力

```
メール: admin@example.jp
パスワード: （管理者から受け取った初期パスワード）
```

**ステップ4**: 「ログイン」ボタンをクリック

### 1-2. パスワード変更（初回ログイン時）

ログイン後、以下のメッセージが表示される場合があります：

```
⚠️ セキュリティ警告
「初期パスワードのまま使用するのは危険です。
 すぐに新しいパスワードに変更してください。」
```

**パスワード変更手順**:

1. 右上のアイコン（👤）をクリック
2. 「設定」→「アカウント情報」を選択
3. 「パスワード変更」をクリック

```
┌─────────────────────────────────────┐
│        パスワード変更                │
│                                     │
│ 現在のパスワード:                   │
│ [                              ]   │
│                                     │
│ 新しいパスワード:                   │
│ [                              ]   │
│                                     │
│ 新しいパスワード（確認）:           │
│ [                              ]   │
│                                     │
│        [  変更する  ]              │
└─────────────────────────────────────┘
```

**安全なパスワード例**:
```
❌ 弱い: password123, school2024
✅ 強い: Cram$School2024@Admin#Secure
```

パスワードの条件：
- 8文字以上
- 大文字・小文字・数字を含む
- 記号（!@#$%）を含む（推奨）

### 2-2. 塾情報フォーム

以下の情報を入力します：

```
【基本情報】

塾の名前*:
（例）〇〇学習塾

塾の説明（50字以内）:
（例）東京都千代田区の個別指導塾。英語・数学を主力科目に、
     大学受験対策から補習まで対応しています。

【所在地】

都道府県*:
（例）東京都

市区町村*:
（例）千代田区

住所（番地）*:
（例）丸の内1-1-1 ビル5F

電話番号*:
（例）0120-123-4567

メールアドレス*:
（例）info@example.jp

【営業時間】

営業開始時刻*:
（例）14:00

営業終了時刻*:
（例）22:00

定休日*:
☐ 月 ☐ 火 ☐ 水 ☐ 木 ☐ 金 ☑ 土 ☑ 日

【ウェブサイト（オプション）】

ウェブサイトURL:
（例）https://example.jp

最寄り駅:
（例）千代田線赤坂見附駅 徒歩3分
```

すべて入力したら、「保存」をクリック。

### 10-2. よくあるトラブルと対応

**トラブル1: ログインできない**

```
症状: 「メールアドレスまたはパスワードが正しくありません」

対応:
1. メールアドレスのタイプミスを確認
2. CapsLock（大文字キー）がONになっていないか確認
3. 「パスワード再設定」をクリック
4. メールで届いた再設定URLからパスワードを変更

それでもダメな場合:
→ support@example.jp に連絡
```

**トラブル2: ファイルがアップロードできない**

```
症状: 「ファイルのアップロードに失敗しました」

対応:
1. ファイル形式を確認
   OK: PDF, JPG, PNG, XLSX
   NG: PSD, EXE, ZIP

2. ファイルサイズを確認
   上限: 50MB（通常の課題は5MB以下）

3. インターネット接続を確認
   → ブラウザの更新ボタンをクリック
   → 再度アップロードを試行

それでもダメな場合:
→ 別のブラウザ（Chrome/Safari）を試す
```

**トラブル3: 採点結果が不正確**

```
症状: 「正解なのに×判定されている」

対応:
1. 採点モードを確認
   • 完全一致採点: スペルを完全に一致させる
   • 通常採点: 軽微な表記ゆれはOK
   • 寛容採点: 内容が正しければOK

2. 採点基準を確認
   → 課題詳細ページで模範解答を再確認

3. 手動での採点修正
   → 誤採点をクリック → 「正解に修正」をクリック

定期的な誤採点は、以下のメールで報告:
→ support@example.jp
```

## トラブルシューティング・FAQ

### よくある質問

**Q1. 生徒が課題を提出したのに、提出状況に反映されていない**

A. 以下を確認してください：
1. 生徒は本当にシステムに登録されているか
2. その生徒は課題対象クラスに属しているか
3. ブラウザをリロード（F5キー）してから確認

それでもダメな場合は、support@example.jp に連絡してください。

**Q2. 講師のパスワードが分からなくなった**

A. 管理者として、以下の対応ができます：
1. 講師管理ページから該当講師をクリック
2. 「パスワード再設定を送信」をクリック
3. 講師のメールに再設定URLが送信される
4. 講師が新しいパスワードを設定

**Q3. 過去の課題・採点記録を削除したい場合**

A. 削除には管理者権限が必要です：
1. 課題詳細ページの「...」をクリック
2. 「削除」をクリック
3. 確認メッセージで「本当に削除する」をクリック

※ 削除後は復旧できません。必要に応じて事前にバックアップを取得してください。

**Q4. 複数の塾を管理している場合**

A. 塾ごとに別々のアカウントを作成してください。
1. 塾A用: admin-a@example.jp
2. 塾B用: admin-b@example.jp

各塾のデータは完全に分離されているので、データ混在の心配はありません。
